# 🐰 Páscoa Website

Um site temático de Páscoa desenvolvido para apresentar uma experiência visual alegre, criativa e envolvente, com foco em um design bonito, clima festivo e navegação agradável.

## ✨ Sobre o projeto

O **Páscoa Website** é uma página web criada com a proposta de transmitir o espírito da Páscoa de forma moderna e visualmente atrativa

## 🎯 Objetivo

Criar uma interface temática de Páscoa que seja:

- Bonita e chamativa
- Leve e agradável de navegar
- Ideal para apresentar conteúdo sazonal de forma criativa

## 🛠️ Tecnologias utilizadas

Este projeto pode incluir tecnologias como:

- PHP
- HTML5
- BootStrap
- CSS3
